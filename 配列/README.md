#  配列

## 配列について
- 記載方法
  ```
  $ month=("A" "B" "C")
  ```
- 要素の取り出し方法
  - 要素の一部
    ```
    $ echo "${month[0]}" 
    ```
  - 全要素
    ```
    $ echo "${month[@]}" 
    ```
- 要素の追加、変更
  ```
  $ month+=("D" "E") or
  $ month=("${month[@]}" "D" "E")
  ```
- 配列の宣言
  ```
  declare -a month=("A" "B" "C")
  ```
- 要素の数と要素の一部の文字数を取得
  ```
  $ echo "${#month[@]}"
  $ echo "${#month[0]}"
  ```
- 配列をつなげる
  ```
  $ array3=("${array1[@]}" "${array2[@]}")
  ```
- 要素の削除
  ```
  $ unset array[0]
  ```
- 要素の一部に取り出し
  ```
  $ month=("A" "B" "C" "D" "E")
  $ echo ${month[@]:2:2} ※2番目の要素から2個取り出し
  C D
  ```
- 一部の要素から一部の文字を取り出し
  ```
  $ month=("A" "Bear" "C" "D" "E")
  $ echo ${month[1]:1:2} ※2番目の要素から2番目から3番目取り出し
  ea
  ```
